Colander data converter documentation
=====================================

The ``colander_data_converter`` Python package provides utilities for converting, validating, and serializing data structures, typically for use with the Colander schema validation library. It includes modules for defining data models, handling common data conversion tasks, and integrating with Colander-based data models and feeds.
